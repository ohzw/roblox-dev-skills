# Roblox Agent Skills

AI エージェントが Roblox Studio MCP で 3D オブジェクトやマップを生成するためのスキル集。

## インストール

```bash
npx skills add ohzw/roblox-dev-skills
```

## スキル一覧

### roblox-design-consultant

実装を始める**前に**、対話を通じて要件を引き出すデザインコンサルタント。リクエストが抽象的だったり、構造的な詳細が不足している場合に使う。

**使うタイミング:**
- 「キッチンを作って」「かっこいいの作って」など、構造が曖昧なリクエスト
- スタイル・スケール・構成要素を決めてから実装したいとき
- `building-3d-objects` が曖昧さを検出して要件定義が必要なとき

**動作の流れ:**
1. 一度に一つだけ質問する（最大 3〜4 ターン）
2. 「どんなスタイルにしますか？」ではなく「モダンとレトロ、どちらにしますか？」のように具体的な選択肢を提示
3. 最終的に `building-3d-objects` に渡せる **Design Brief** を出力して終了

---

### building-3d-objects

`mcp__roblox__run_code` を使って、Roblox Studio に 3D オブジェクト・プロップ・部屋スケールの構造物を生成する。

**使うタイミング:**
- 「○○を作って」「○○を置いて」「build me a ___」
- 家具・乗り物・門・塔・デコレーション・部屋インテリアなど
- 結果が単一の `Model` にまとまるスコープのもの

**カバーする内容:**
- 4 フェーズのビルドプロセス: Assess（設計）→ Plan（座標系）→ Build（ジオメトリ）→ Verify（検証）
- 曲面・穴・中空形状のための CSG (Union/Subtract) パターン
- 複数の MCP 呼び出しをまたいだ座標管理
- アンカー忘れやジオメトリエラーを検出する自動検証スクリプト
- バリアントセット対応（例：「5種類の剣」）

---

### building-maps

Roblox Studio に、マップ・アリーナ・ロビーなど大規模なマルチゾーン環境を生成する。

**使うタイミング:**
- 「マップを作って」「バトルエリアを作って」「ロビーを作りたい」
- 複数のゾーンや広域のレイアウトが必要なとき
- 空間設計・ゾーン計画が伴うリクエスト

**カバーする内容:**
- 6 フェーズのビルドプロセス: Layout → Ground → Zone Shells → Landmarks → Fill → Environment
- 浮動小数点誤差の蓄積を防ぐ Origin 基準の相対座標系
- ワークスペースを整理するフォルダ階層
- Lighting サービス・Atmosphere・SpawnLocation を含む環境設定
- 自動検証スクリプト
